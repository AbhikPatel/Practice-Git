Welcome to 21 Card Game

Note: If you have played RESIDENT EVIL 7: BIOHAZARD then skip the Description and Smash that Start Game Button.
  • First of all this game is a 2-Player game so first step for you is get a friend.
  • Enter your Names.
  • Then Start Game.
  • When the Game is Started each player will have a card which contains a number(Which will be hidden).
  • Then when the turn comes you will have 2 Chioces REQUEST A CARD or PASS.
  • REQUEST A CARD will give a card with a number.
  • PASS means no action is taken and passes to another Player.
  • When both the players passes then the final result is done.
  • Final Result means all the number of the card are add on and final result should be under 21.
  • If the summation goes more than 21, then the Player is Eliminated.
  • But if both the Players has less then 21 then its depend upon who has more number.
  • 100 Baat ki 1 Baat :- Get more number but don't go more than 21.
